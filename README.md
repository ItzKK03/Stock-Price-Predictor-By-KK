📈 AI Stock Price Predictor

A comprehensive, AI-powered web application that forecasts stock closing prices using a sophisticated hybrid model. This project combines Long Short-Term Memory (LSTM) deep learning for time-series analysis with Natural Language Processing (NLP) for market sentiment, providing a more holistic prediction.

The entire application is built with a clean, interactive user interface using Streamlit.

🚀 Core Features

LSTM-Based Time-Series Forecasting: Utilizes a deep learning LSTM model to analyze historical price patterns and trends over the last 60 days.

NLP Sentiment Analysis: Employs a pre-trained FinBERT model (from Hugging Face transformers) to analyze the sentiment of recent news headlines, integrating market emotion as a predictive feature.

Technical Indicator Analysis: Automatically calculates and feeds key technical indicators (RSI, MA20, MA50, Volume) into the model as additional features.

Interactive Streamlit UI: A modern, responsive web dashboard built with Streamlit to fetch live data, make predictions, and visualize the results.

Modular & Scalable: The codebase is organized into distinct modules (data_collection, feature_engineering, sentiment_analysis, model_utils) for easy maintenance and future extensions.

🛠️ Tech Stack

Web Framework: Streamlit

Deep Learning: TensorFlow / Keras

NLP: Hugging Face Transformers (using FinBERT)

Data Analysis: Pandas, NumPy

Stock Data: yfinance

Technical Indicators: pandas_ta

Plotting: Plotly

Model Serialization: Joblib

⚡ How to Run

This repository includes the pre-trained model files (.keras, .joblib) so you can run the app immediately.

1. Clone the Repository

git clone [https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git)
cd YOUR_REPOSITORY_NAME


2. Set Up a Virtual Environment

It's highly recommended to use a virtual environment to manage dependencies.

# Windows
python -m venv venv
.\venv\Scripts\activate


3. Install Dependencies

pip install -r requirements.txt


4. Run the Streamlit App

streamlit run app.py


Your app will now be running and accessible at http://localhost:8501 in your browser.

(Optional) Re-train the Model

If you want to train the model from scratch on your own machine:

python train_model.py


This will generate new .keras and .joblib files based on the latest available data.
